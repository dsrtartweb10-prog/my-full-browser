## ✨ Features
- 🗂️ **Tabbed Browsing**
- ⭐ **Bookmarks**
- 🕘 **History**
- ⬇️ **Download Manager**
- 🔄 **Auto-update**
- 🖥️ **Cross-platform** (Windows, macOS, Linux)
